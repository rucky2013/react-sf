# 运行方式

全局安装Node.js、webpack、webpack-dev-server

开始爬虫服务器
``` sh
$ cd react-sf
$ cd server
$ npm install
$ node app
```
开启app
``` sh
$ cd react-sf
$ cd app
$ npm install
$ npm run dev
```


#开发运行npm run dev
#打开 http://localhost:8080/

#打包出来运行npm run build
